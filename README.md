**Installation pour un fonctionnement local**
---------------------------------------------

* Installation de l'environnement Anaconda

Cloner le répertoire puis exécuter à l'intérieur du répertoire : 
```bash
conda env create
conda activate BAP-Bot
```
* Lancement du Bot
```bash
python module.py
```
